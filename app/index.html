GeneMO Web Example

This is an example of using GeneMO library for running genetic algorithm in web browser. GeneMO executes program in small asynchronous chunks to avoid blocking browser's main thread. Source code of this example can be found in GeneMO Web Example Github repository.

This example solves Traveling Salesman Problem. It searches for the shortest path to visit 17 cities, given distances between each pair of these cities.
